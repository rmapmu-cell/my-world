## 建议规格

- **比例**：竖版 2:3（比如 500 × 750 像素）—— 卡片是竖的，横图会被裁掉两边
- **格式**：`.jpg`（想换 `.png` / `.webp` 告诉我改一行代码）
- **大小**：单张 300KB 以下最好（网页加载快）

## 没放的图会怎样？

**自动 fallback 到 CSS 场景插画** —— 你可以先放两三张试试，其他慢慢补。不会出现空白卡片。
